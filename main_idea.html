Qual nossa ideia?

A pandemia, além da economia, afetou principalmente a saúde mental e a educação dos alunos, então, seguindo a mesma lógica, como poderíamos apresentar o mesmo ensino tradicional para alunos com problemas individuais psicológicos ou dificuldades de aprendizagem? É impossível a administração das escolas atender a individualidade de cada um, a Project A visa uma solução para o problema, prevê um sistema com capacidades de analisar o desempenho individual em notas dos alunos e de recomendar, para a administração escolar, aulas de reforços em disciplinas com baixo desempenho.

Como pode ser colocado em prática?

1. Obter os dados a partir do RCO.

2. Calcular a média de notas da escola, comparar a média com a nota dos alunos.

3. Identificar os alunos com média baixa.

4. Comparar a média da nota escolar entre as escolas próximas, afim de obter feedback.

5. Recomendar aulas de reforços para os alunos.
